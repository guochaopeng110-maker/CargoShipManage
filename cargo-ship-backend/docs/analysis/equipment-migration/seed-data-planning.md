## 数据生成策略

### 时序数据生成原则
- 每个监测点生成**3条数据**
- 时间戳: 最近7天内随机分布
- 数据质量: 
  - 第1条: normal (正常值)
  - 第2条: normal (正常值)
  - 第3条: abnormal (触发阈值告警，用于测试告警功能)

### 阈值配置生成原则
- 完全基于 `docs/data/` 中的表格数据
- 包含所有故障等级（1级、2级、3级）
- 填充完整的 `fault_name` 和 `recommended_action` 字段
