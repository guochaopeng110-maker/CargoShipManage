## 附录：监测点命名规范

### 1. 命名原则

- 使用中文自然语言，简洁明确
- 与客户需求文档保持一致
- 区分监测点与告警规则
  - **监测点**: 物理测量值（如"环境温度"、"单体电压"）
  - **告警规则**: 基于监测点的告警名称（如"环境温度过高-一级"、"单体压差大-二级"）

### 2. 数据类型说明

- **number**: 数值型数据，包括电压、电流、温度、压力等
- **boolean**: 布尔型数据，表示开关状态、故障状态等

### 3. 单位规范

| 物理量 | 单位 | 说明 |
|--------|------|------|
| 电压 | V | 伏特 |
| 电流 | A | 安培 |
| 温度 | °C | 摄氏度 |
| 功率 | kW | 千瓦 |
| 转速 | rpm | 转/分钟 |
| 频率 | Hz | 赫兹 |
| 压力 | MPa | 兆帕 |
| 电阻 | kΩ | 千欧 |
| 百分比 | % | 百分比 |
| 水位 | mm | 毫米 |

## 使用说明

1. **前端开发**: 根据此文档设计监测点数据展示界面
2. **后端开发**: 确保API返回的监测点名称与此文档一致
3. **数据导入**: Excel/CSV模板的"监测点"列必须使用此文档中的标准名称
4. **告警配置**: 阈值配置时的 `monitoringPoint` 字段必须使用此文档中的标准名称
5. **测试数据**: 生成测试数据时使用此文档中定义的监测点

**注意**: 监测点名称是系统的核心元数据,任何修改都需要同步更新代码、数据库和文档。
